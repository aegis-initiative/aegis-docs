# Assets

This directory contains SVG logos, wordmarks, and other static assets used by the site.

## Planned Assets

- `aegis-wordmark.svg` — AEGIS wordmark (from repo assets/)
- `aegis-wordmark-dark.svg` — Dark variant
- `aegis-wordmark-light.svg` — Light variant
- `aegis-logo.svg` — AEGIS shield logo

Assets in the repo root `assets/` directory are the canonical source files.
Assets here in `src/assets/` are optimized copies used by Astro's image pipeline.
