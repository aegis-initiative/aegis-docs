# Plugins

This directory will contain remark and rehype plugins for MDX processing.

## Planned Plugins (from aegis-constitution)

### remark-asides

Transforms `:::note`, `:::tip`, `:::caution`, and `:::danger` fenced directives
into styled callout blocks. This plugin is used in aegis-constitution and will be
shared via the common design system.

### rehype-autolink-headings

Adds anchor links to headings for deep-linking to document sections.

These plugins will be extracted from aegis-constitution when the shared design
system package is built.
